# BertlClaw Indexing Monitoring Plan

## Ziel
Prüfen, wann Google BertlClaw indexiert und wie sich die Brand-Suche entwickelt.

## Täglich für die nächsten 3-7 Tage prüfen

### 1. Google Search Console
- Status der Sitemap
- Seitenindexierung
- URL-Prüfung für:
  - /
  - /services.html
  - /use-cases.html
  - /landingpages.html
  - /ki-fuer-selbststaendige.html

### 2. Google-Suche
Suche nach:
- `BertlClaw`
- `site:bertlclaw.github.io/bertlclaw BertlClaw`
- `site:bertlclaw.github.io/bertlclaw`

### 3. Bing-Suche
Suche nach:
- `BertlClaw`

## Woran Fortschritt erkennbar ist
- erste Seiten erscheinen bei `site:`-Suche
- Search Console meldet erkannte/indexierte Seiten
- die Brand-Suche `BertlClaw` zeigt GitHub oder GitHub Pages Ergebnisse
- Impressionen in Search Console steigen an

## Nächster sinnvoller SEO-Schritt, wenn nach einigen Tagen wenig passiert
- zusätzliche Brand-Signale setzen
- externe Erwähnungen / Profile verlinken
- GitHub-Profil weiter ausbauen
